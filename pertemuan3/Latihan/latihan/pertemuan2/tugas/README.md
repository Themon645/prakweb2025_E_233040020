# Tugas: CRUD sederhana (MVC)

Lokasi: `pertemuan2/tugas/mvc`

Cara menjalankan (menggunakan PHP built-in server):

1. Buka PowerShell di root project.
2. Jalankan server:

```powershell
& 'C:\laragon\bin\php\php-8.1.10-Win32-vs16-x64\php.exe' -S localhost:8000 -t pertemuan2/tugas/mvc
```

3. Buka browser ke `http://localhost:8000/index.php`

Fitur:
- Daftar produk (Read)
- Tambah produk (Create)
- Edit produk (Update)
- Hapus produk (Delete)

Data disimpan sederhana di `pertemuan2/tugas/mvc/data/produk.json`.
